# BidGenie Platform

BidGenie is a live-data-driven auction system featuring real-time backend telemetry, a secure task management system, machine learning-driven price predictions, and risk assessments.

## Project Structure

- `frontend/`: The user interface, featuring a dynamic cyberpunk design and real-time dashboards.
- `backend/`: The core server handling authentication, auction logic, and API endpoints.
- `ml-service/`: Machine learning integration for price prediction and fraud detection based on auction datasets.
